## Description

SoDeBe aka Software Design Backend

## Installation

```bash
$ pnpm install
```

Then create a `.env` file in the root level of the project, and paste the content that we pinned on our Messenger group chat.

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
